This package defines the common, core types used by the Mesh Configuration Protocol.

Metadata

Metadata information that all resources within the Mesh Configuration Protocol must have.

Field Type Description
name string

The name of the resource. It is unique within the context of a resource type and the origin server of the resource. The resource type is identified by the TypeUrl of the contents field of the Resource message.

Resource

A configuration resource as transferred via the Mesh Configuration Protocol. Each resource is made up of common metadata, and a type-specific payload.

Field Type Description
metadata Metadata

Common metadata describing this resource.
contents google.protobuf.Any

The type-specific payload for this resource.
