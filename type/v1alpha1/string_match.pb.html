StringMatch

Describes how to match a given string. Match is case-sensitive.

Field Type Description Required
exact string (oneof)

exact string match

 No
prefix string (oneof)

prefix-based match

 No
regex string (oneof)

RE2 style regex-based match (https://github.com/google/re2/wiki/Syntax).

 No
